## Expense Calculator App

A simple, user-friendly expense calculator built with HTML, CSS, and JavaScript.

### Features
- ✅ Add expenses with name, amount, and category
- ✅ Multiple expense categories (Food, Transportation, Entertainment, etc.)
- ✅ Real-time total calculation
- ✅ Delete individual expenses
- ✅ Data persistence using localStorage
- ✅ Responsive design with modern UI

### How to Use
1. Open `expense-calculator.html` in your web browser
2. Enter an expense name, amount, and select a category
3. Click "Add Expense" to add it to your list
4. View your total expenses in the Summary section
5. Click "Delete" on any expense to remove it

All your expenses are automatically saved in your browser's localStorage!
