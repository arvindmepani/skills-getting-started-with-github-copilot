## 💰 Expense Tracker App

This repository now includes a fully functional **Expense Tracker** application built with React!

### Features
- ✨ Beautiful, sparkling UI with gradient designs and animations
- 💰 Track expenses with categories (Food, Transport, Entertainment, etc.)
- 📊 Real-time total expense calculations
- 🗑️ Easy expense management with delete functionality
- ✅ Comprehensive unit tests with Vitest and React Testing Library
- 📱 Responsive design for all devices

### Quick Start

```bash
cd expense-generator
npm install
npm run dev
```

Visit `http://localhost:5173` to see the app in action!

### Run Tests

```bash
cd expense-generator
npm run test:run
```

For more details, see the [Expense Generator README](./expense-generator/README.md).
